
# 🎮 Tetris Game in C++ using SFML

This is a classic **Tetris** game built with **C++17** and **SFML 2.6.2**. It features a clean object-oriented design, game logic for seven different Tetrimino shapes, scoring, pause menu, and more.

---

## 📌 Features

- 🧱 All 7 Tetrimino shapes (`I`, `O`, `T`, `L`, `J`, `S`, `Z`)
- 🎨 Shape colors for easy distinction
- ⏸️ Pause and resume functionality
- 🔢 Score tracking system
- 👤 Player name input at game start
- ➕ Next piece preview
- 🎯 Soft drop and instant hard drop
- 💻 Developed with C++ and SFML 2.6.2

---

## 🛠️ Requirements

- C++17 or later
- [SFML 2.6.2]

---


### 🪟 Windows (Visual Studio)
1. Clone the repo.
2. Open the `.sln` file in Visual Studio.
3. Make sure SFML is properly configured (link/include dirs).
4. Build and run.

---

## 🎮 Controls

| Key         | Action               |
|-------------|----------------------|
| ← / →       | Move left/right      |
| ↓           | Soft drop            |
| ↑           | Rotate               |
| P           | Pause / Resume       |
| E           | Exit to menu         |

---

## 🧱 Code Structure

Piece abstract class -> inherited tetrimino class
Board class (includes all functionalities)
